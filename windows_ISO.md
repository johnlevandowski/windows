Windows ISO
===========

Use Windows Media Creation Tool when actually logged in to windows admin account.  
Possibly need to run from somewhere other than downloads folder.  
ISO from Windows Media Creation Tool is more up to date.  

Use rufus (portable exe) to download latest windows ISO  
Allow ov-df.microsoft.com in pi-hole  
SELECT > DOWNLOAD > Click DOWNLOAD > Windows 11 > English (United States) > Download in browser (use URL from log file if needed)  

To download windows ISO from microsoft website, close browser, disable pihole, then download
